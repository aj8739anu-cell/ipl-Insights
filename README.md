# ipl-Insights
Overview

This project focuses on analyzing Indian Premier League (IPL) match data using Python, Pandas, and data visualization libraries.
The goal is to extract meaningful insights such as team performance, toss impact, player achievements, and match trends.

Objectives
Analyze IPL match data
Identify top-performing teams and players
Understand toss impact on match outcomes
Explore scoring and bowling trends
Visualize key insights using graphs
Tech Stack
Python
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib & Seaborn – Data visualization
Dataset
File used: IPL.csv
Contains match-level IPL data including:
Match winner
Toss winner & decision
Player of the match
Top scorer & high score
Winning method (runs/wickets)
Key Analysis Performed
1. Team with Most Match Wins
Counted total wins per team
Visualized using bar chart
2. Toss Decision Trends
Analyzed how often teams choose:
Bat
Field
3. Toss Winner vs Match Winner
Calculated percentage of matches where:
Toss winner = Match winner
4. Winning Type Analysis
Compared wins by:
Runs
Wickets
5. Player of the Match Analysis
Identified top 10 players with most awards
6. Top Scorers
Found players with highest cumulative scores
7. Best Bowling Performances
Analyzed standout bowling figures
Visualizations
Bar plots
Count plots
Horizontal bar charts

These help in understanding trends clearly and effectively.

How to Run the Project
Clone the repository:
git clone https://github.com/your-username/ipl-data-analysis.git
Navigate to the folder:
cd ipl-data-analysis
Install dependencies:
pip install pandas numpy matplotlib seaborn
Run the notebook:
jupyter notebook
